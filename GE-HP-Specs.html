HP Spectre x360 14

SPECIFICATION

  • Processor: Intel Core i7-1165G7
  • Graphics: Intel Iris Xe Graphics
  • Memory: 16GB RAM
  • Storage: 1TB SSD
  • Display: 13.5" FHD touch display

FEATURES

  • Performance: The Intel Core i7 processor and Iris Xe graphics deliver strong performance for everyday tasks and some creative work.
  • 2-in-1 Design: Versatile 2-in-1 design allows it to be used as both a laptop and a tablet, with a 360-degree hinge.
  • Battery Life: Excellent battery life, making it ideal for long work sessions and travel.
  • Build Quality: High-quality materials and design, with a sleek and stylish look.
  • Security: Features like a fingerprint reader and webcam kill switch enhance security and privacy.
